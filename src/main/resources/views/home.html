Mu Server

Java's modern web server

Quick start

Add one dependency...

{% include "incs/maven.html" %}

...and say hello:

import io.muserver.*;
public class Hello {
    public static void main(String[] args) {
        MuServer server = MuServerBuilder.httpServer()
            .addHandler(Method.GET, "/", (request, response, pathParams) -> {
                response.write("Hello, world");
            })
            .start();
        System.out.println("Started server at " + server.uri());
    }
}

Features

  • Simple static resource handling
  • Extremely simple HTTPS configuration
  • Server Sent Events
  • Filters
  • Routing with path templates
  • JAX-RS Rest support with no extra dependencies
  • Automatic documentation for rest services
  • Simple CORS support for JAX-RS
  • File Upload support
  • Trivial support for streaming data
  • Support for Range requests
  • Optional async handling
  • Path contexts
  • Header parsers for some common header types
  • Statistics
  • GZIP
  • Various useful utilities

For more info see https://github.com/3redronin/mu-server or check out the Java docs.
